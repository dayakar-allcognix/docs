## How to Use

### Step 1: Access the Tool

1. Navigate to the **Text to Image** section in your dashboard
2. You'll see the generation interface with a text input field
3. Notice the microphone icon for voice input

### Step 2: Write Your Prompt

<Steps>
  <Step title="Basic Prompt">
    Start with a simple description:
    ```
    A beautiful sunset over mountains
    ```
  </Step>
  
  <Step title="Enhanced Prompt">
    Add more details for better results:
    ```
    A photorealistic sunset over snow-capped mountains with a crystal clear lake reflecting the peaks, soft golden lighting, cinematic composition, high resolution
    ```
  </Step>
  
  <Step title="Voice Input">
    Click the microphone icon and speak your prompt
    - Supports multiple languages
    - Automatically converts speech to text
    - Great for hands-free operation
  </Step>
</Steps>

### Step 3: Configure Settings

<AccordionGroup>
  <Accordion title="Aspect Ratio">
    Choose the perfect dimensions for your use case:
    - **Square (1:1)**: Perfect for social media posts
    - **Landscape (16:9)**: Great for banners and presentations
    - **Portrait (9:16)**: Ideal for mobile content
    - **Custom**: Set your own dimensions
  </Accordion>
  
  <Accordion title="AI Model Selection">
    Select the best model for your needs:
    - **Omni Photo T2i V1**: Balanced quality and speed
    - **Omni Photo T2i V2**: Enhanced quality and detail
  </Accordion>
  
  <Accordion title="Style Options">
    Choose the artistic style:
    - **Photorealistic**: Life-like images
    - **Artistic**: Creative and stylized
    - **Minimalist**: Clean and simple
    - **Cinematic**: Movie-like quality
  </Accordion>
</AccordionGroup>

### Step 4: Generate and Refine

1. Click **Generate** to create your image
2. Wait for the AI to process (usually 10-30 seconds)
3. Review the result and make adjustments if needed
4. Use **Regenerate** to try variations
5. **Save** your favorite results

## Prompt Writing Tips

### Effective Prompt Structure

<Accordion title="Subject + Style + Quality">
**Format**: `[Subject] in [Style] style, [Quality modifiers]`

**Example**: 
```
A futuristic city skyline in cyberpunk style, neon lights, high resolution, detailed architecture
```
</Accordion>

<Accordion title="Lighting and Composition">
Include lighting and composition details:
- **Lighting**: Soft lighting, dramatic shadows, golden hour, studio lighting
- **Composition**: Close-up, wide shot, aerial view, rule of thirds
- **Mood**: Serene, energetic, mysterious, peaceful
</Accordion>

<Accordion title="Quality Modifiers">
Add these keywords for better results:
- High resolution, detailed, professional
- Sharp focus, crisp details
- 4K, ultra-high definition
- Masterpiece, award-winning
</Accordion>

### Common Prompt Patterns

<AccordionGroup>
  <Accordion title="Photography Style">
    ```
    Professional photograph of [subject], [lighting], [composition], high resolution, detailed
    ```
  </Accordion>
  
  <Accordion title="Artistic Style">
    ```
    [Art style] painting of [subject], [colors], [mood], artistic, creative
    ```
  </Accordion>
  
  <Accordion title="Product Visualization">
    ```
    Product shot of [item], clean background, professional lighting, commercial photography
    ```
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Poor Quality Results">
    **Solutions:**
    - Add more descriptive details to your prompt
    - Try different style options
    - Use quality modifiers (high resolution, detailed)
    - Experiment with different AI models
  </Accordion>
  
  <Accordion title="Voice Input Not Working">
    **Solutions:**
    - Check microphone permissions
    - Ensure stable internet connection
    - Try refreshing the page
    - Use text input as alternative
  </Accordion>
  
  <Accordion title="Generation Taking Too Long">
    **Solutions:**
    - Check your internet connection
    - Try a simpler prompt
    - Switch to V1 model for faster generation
    - Contact support if issues persist
  </Accordion>
</AccordionGroup>

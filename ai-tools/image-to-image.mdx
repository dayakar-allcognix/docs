## Overview

The Image to Image tool, also known as Depth to Image, allows you to transform existing images using AI technology. Upload any image and use custom prompts to create style transfers, enhance photos, or apply creative effects while maintaining the original composition and depth information.

## How to Use

### Step 1: Upload Your Image

<Steps>
  <Step title="Select Image">
    Click **Upload Image** or drag and drop your file
    - Supported formats: JPG, PNG, WebP
    - Maximum file size: 10MB
    - Recommended resolution: 512x512 to 2048x2048
  </Step>
  
  <Step title="Image Preview">
    Review your uploaded image
    - Check image quality and composition
    - Ensure the subject is clearly visible
    - Verify the image loads correctly
  </Step>
</Steps>

### Step 2: Write Your Transformation Prompt

<AccordionGroup>
  <Accordion title="Style Transfer Prompts">
    **Artistic Styles:**
    ```
    Transform into oil painting style, rich colors, brush strokes visible
    ```
    
    **Photography Styles:**
    ```
    Convert to vintage film photography, warm tones, grainy texture
    ```
  </Accordion>
  
  <Accordion title="Enhancement Prompts">
    **Quality Improvement:**
    ```
    Enhance details, improve lighting, sharpen focus, professional quality
    ```
    
    **Mood Changes:**
    ```
    Change to dramatic lighting, cinematic atmosphere, golden hour
    ```
  </Accordion>
  
  <Accordion title="Creative Transformation">
    **Environmental Changes:**
    ```
    Transform to winter scene, add snow, cold atmosphere
    ```
    
    **Fantasy Elements:**
    ```
    Add magical elements, glowing effects, fantasy atmosphere
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Configure Advanced Settings

<AccordionGroup>
  <Accordion title="Transformation Strength">
    Control how much the AI transforms your image:
    - **Low (0.3-0.5)**: Subtle changes, preserves original
    - **Medium (0.5-0.7)**: Balanced transformation
    - **High (0.7-0.9)**: Strong transformation, more creative
  </Accordion>
  
  <Accordion title="Depth Preservation">
    Maintain spatial relationships:
    - **Preserve Depth**: Keep original depth information
    - **Enhance Depth**: Improve depth perception
    - **Creative Depth**: Modify depth for artistic effect
  </Accordion>
  
  <Accordion title="Style Consistency">
    Ensure consistent style application:
    - **Uniform Style**: Apply style evenly across image
    - **Selective Style**: Focus on specific areas
    - **Gradient Style**: Gradual style transition
  </Accordion>
</AccordionGroup>

### Step 4: Generate and Refine

1. Click **Transform Image** to start processing
2. Wait for AI processing (15-45 seconds depending on complexity)
3. Review the transformed result
4. Use **Adjust Settings** to fine-tune
5. **Save** your favorite transformations

## Quality Optimization

### Input Image Requirements

<Accordion title="Best Practices">
- **Resolution**: Use images between 512x512 and 2048x2048 pixels
- **Format**: JPG for photos, PNG for graphics with transparency
- **Quality**: Avoid heavily compressed or low-quality source images
- **Composition**: Ensure clear subject and good lighting
</Accordion>

### Output Quality Settings

<AccordionGroup>
  <Accordion title="Resolution Options">
    - **Standard**: 1024x1024 (faster processing)
    - **High**: 1536x1536 (balanced quality/speed)
    - **Ultra**: 2048x2048 (maximum quality)
  </Accordion>
  
  <Accordion title="Processing Quality">
    - **Fast**: Quick processing, good quality
    - **Balanced**: Standard processing time and quality
    - **High**: Longer processing, superior quality
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Poor Transformation Results">
    **Solutions:**
    - Adjust transformation strength
    - Try different prompt approaches
    - Use higher quality source images
    - Experiment with different AI models
  </Accordion>
  
  <Accordion title="Style Not Applied Consistently">
    **Solutions:**
    - Use more specific style descriptions
    - Adjust style consistency settings
    - Try selective masking
    - Use reference images for style
  </Accordion>
  
  <Accordion title="Loss of Important Details">
    **Solutions:**
    - Reduce transformation strength
    - Use enhancement-focused prompts
    - Preserve depth information
    - Try selective transformation
  </Accordion>
</AccordionGroup>

## Best Practices

<Accordion title="Prompt Writing">
